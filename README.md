# LuminaPack AI - Configuración del Proyecto

¡Bienvenido al código fuente de LuminaPack AI!

## ⚠️ Requisito Importante: Node.js

He detectado que **Node.js** no está instalado en tu sistema (o no está en el PATH), lo que impidió la inicialización automática.

Para ejecutar este proyecto, necesitas:

1.  **Descargar e Instalar Node.js**:
    *   Ve a [nodejs.org](https://nodejs.org/).
    *   Descarga la versión **LTS (Recomendada)**.
    *   Instálalo (sigue los pasos del instalador, asegúrate de marcar "Add to PATH").
    *   **Reinicia tu terminal** (o VS Code) después de instalar.

## Cómo Ejecutar el Proyecto

Una vez tengas Node.js instalado:

1.  Abre una terminal en esta carpeta (`C:\Users\Note\Desktop\Nueva carpeta`).
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Abre el link que aparece (usualmente `http://localhost:5173`).
